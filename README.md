## 🧠 What You Will Learn

This tutorial covers the essential Git commands:

```bash
git status
git add .
git commit -m "Your message"
git pull origin main
git push origin main
```

### Explanation:

* **git status** → Check changes in your project
* **git add .** → Add all files to staging
* **git commit -m "message"** → Save changes locally with a comment
* **git pull origin main** → Get latest updates from GitHub
* **git push origin main** → Upload your changes to GitHub

## 🚀 How to Use (Git Workflow)

### 🟢 First Time Setup

```bash
git init
git remote add origin https://github.com/username/repository.git
git branch -M main
```

---

### 🔵 Save Changes (Daily Workflow)

```bash
git status
git add .
git commit -m "Update: describe your changes"
```

---

### 🟡 Sync with GitHub

```bash
git pull origin main
git push origin main
```

## 🧑‍🏫 Instructor Notes

* Always **pull before push** when working in teams
* Write **clear commit messages**
* Commit **frequently** to track progress
* Use `git status` before committing
